# Architecture
## Build & Delivery
- build, CI/CD: fastlane
- store delivery: fastlane
- OTA: Hot Updater (https://github.com/gronxb/hot-updater)

## Infra
- navigation: React Navigation
- State management: Redux Toolkit (https://redux-toolkit.js.org/)
- Data fetch: RTK Query (https://redux-toolkit.js.org/)
- Cache normalization deduplication: none
- Monitoring: Sentry
- Showcase components: Storybook
- UI: GluestackUI

## Features
- Camera: react-native-vision-camera
- Location: @react-native-community/geolocation
- Notifications: notifee + firebase/messaging
- SQLite: op-sqlite
- File System: react-native-fs
- Image Picker: react-native-image-picker
- Secure Storage: react-native-keychain
- Audio/Video: react-native-video + react-native-track-player

## Backend
- Backed as a Service: Supabase
